Конфигуратор¶

Конфигуратор - отдельное приложение для работы с моделью технического объекта в платформе «МПК-Пересвет». На данный момент поддерживается работа с данными сущностями:

  • Объект

  • Тег

  • Тревога

  • Хранилище данных

Конфигуратор упрощает работу с платформой и позволяет осуществлять CRUD операции с узлами в иерархии. Он использует стандартное API платформы и отправляет http запросы для реализации CRUD методов.

В качестве технологического стека был использован язык TypeScript и фреймворк Vue.

Содержание:

  • Используемые термины
  • Инструкция
    • Подключение к платформе МПК-Пересвет
    • Просмотр иерархии технического объекта
    • Добавление нового узла иерархии
    • Удаление узла иерархии
    • Редактирование узла иерархии
  • Режим отладки
